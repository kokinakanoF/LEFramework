# LEFramework
（実装途中）<br>
<br>
UnrealEngineでUIやシステムの実装を気軽に行えるようにするためのプラグインです<br>
基本的なUIを親クラスの設定で賄って、楽に作成が行えるようにします。<br>


＜基本設計＞<br>
・インタラクション可能なUIは、状態変化にアニメーションを使用することを前提としています<br>
・継承を繰り返しながらクラスを作成することで、必要な機能だけを含んだシンプルなウィジェットブループリントを作成できます<br>
・関数はC++、BPで継承可能な形にし、カスタマイズ性を高めています<br>
・根本的な改造が必要な場合は、サブクラスの作成ではなく、直接変更を加えてください<br>
